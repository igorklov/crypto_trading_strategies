## Формат 1: Простой

🚀 Вышла статья 3/12 цикла «LLM‑агенты для трейдинга»!

Научим агента анализировать рынок, строить графики и отправлять умные уведомления в Telegram. Всё на Python + OpenClaw.

🧵↓

📊 **Технические индикаторы в Python:**
- RSI (перекупленность/перепроданность)
- MACD (трендовый момент)
- Bollinger Bands (волатильность)
- SMA/EMA (скользящие средние)

Библиотека `ta` делает расчёт в 3 строки кода.

📈 **Автоматическая визуализация:**
Агент строит профессиональные графики с помощью `matplotlib`:
- Цена + скользящие средние
- RSI с уровнями 30/70
- MACD с гистограммой

Графики сохраняются каждые 5 минут.

🤖 **Умные алерты в Telegram:**
Настраиваем условия:
- Изменение цены >2% за 5 мин
- RSI выходит из зоны 30–70
- MACD бычий/медвежий кроссовер
- Цена выходит за Bollinger Bands

При срабатывании — мгновенное уведомление.

⚙️ **Интеграция в единый монитор:**
Скрипт `advanced_monitor.py` объединяет:
- Сбор цены с Binance
- Расчёт индикаторов
- Проверку условий
- Отправку в Telegram
- Логирование в CSV

Работает 24/7 как systemd‑служба.

🔧 **Что получает трейдер:**
- Автономный аналитик на своём сервере
- Освобождение от ручного мониторинга
- Документирование всех сигналов
- Мгновенные оповещения о важных событиях
