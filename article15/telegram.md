1/11 🤔 Зачем трендовые линии в автоматической системе?

Индикаторы (RSI, MACD) либо опережающие, либо запаздывающие. Они следуют за ценой.

Трендовые линии дают другое:

• Сигнал слома структуры — моментально
• Обнаружение смены тренда — раньше, чем SMA или MACD
• Фильтр сигналов — пробой uptrend блокирует buy-сигналы
• Динамические уровни для TP и SL
• Контекст для LLM: «цена пробила 3-месячную трендовую» — это куда информативнее, чем «RSI=30»

➖➖➖

2/11 🎯 Как найти опорные точки (swing highs/lows)

Основа трендовой линии — экстремумы. Нам нужно найти значимые максимумы и минимумы.

Алгоритм:

```python
def find_swing_points(highs, lows, window=5):
    length = len(highs)
    swing_highs = np.full(length, np.nan)
    swing_lows = np.full(length, np.nan)
    
    for i in range(window, length - window):
        if highs[i] == max(highs[i-window:i+window+1]):
            swing_highs[i] = highs[i]
        if lows[i] == min(lows[i-window:i+window+1]):
            swing_lows[i] = lows[i]
    
    return swing_highs, swing_lows
```

window=5 на 15m = ~6 часов истории. window=3 на 4H = 12 часов.
Чем больше window → меньше точек, но они значимее.

3/11 📐 Построение линий и проверка касаний

Линия строится по ДВУМ точкам. Третья точка — подтверждение значимости.

Алгоритм:
1. Перебираем все пары swing-точек
2. Для каждой пары считаем наклон и intercept
3. Считаем, сколько точек лежат в пределах 3% от линии
4. Чем больше «касаний» → тем надёжнее линия
5. Сортируем — возвращаем топ-3

Одна линия с 4+ касаниями — сильный уровень. Линия с 2 касаниями — шум.

➖➖➖

4/11 🚨 Детекция пробоя (breakout)

Самый ценный сигнал — когда цена пересекает трендовую линию.

Два основных типа:

🟢 Bullish breakout: цена была НИЖЕ downtrend → стала ВЫШЕ
   = медвежий тренд сломлен, ожидаем рост

🔴 Bearish breakout: цена была ВЫШЕ uptrend → стала НИЖЕ
   = бычий тренд сломлен, ожидаем падение

Ключевой момент: пробой засчитывается, когда цена пересекает линию, а не просто касается её.

➖➖➖

5/11 🔍 Фильтрация ложных пробоев (главная проблема)

До 60% пробоев — ложные. Цена «протыкает» линию и возвращается.

Наши фильтры:
1. Моментум > 0.5× ATR — цена должна реально пробить
2. Объём > 1.5× среднего — пробой на объёме
3. Сила линии > 60% — линия с 3+ касаниями
4. Нормализованный моментум > 1.0× ATR

Хотя бы 2 из 4 критериев = сигнал считается валидным.

7/11 🎛 Комбинирование сигналов: TA + TL + FA

Финальное решение учитывает три слоя с разными весами:

Технические индикаторы — 40%
Слой: RSI, ADX, BB

Трендовые линии — 35%
Слой: пробои, текущий тренд

Фундаментальный анализ (LLM) — 25%
Слой: новости, сентимент

Пример:

RSI=25 (buy) + пробой downtrend + позитивный FA = BUY ✅
RSI=75 (sell) + пробой uptrend + негативный FA = SELL ✅
RSI=25 (buy) + uptrend intact + негативный FA = HOLD ⏸ (FA блокирует)

➖➖➖

8/11 🤖 LLM для контекстного анализа пробоев

Не все пробои одинаковы. LLM оценивает контекст:

Пример: BTC пробивает uptrend на 4H

LLM: {"score": 7, "explanation": "Пробой подтверждён объёмом (+80%), линия с 4 касаниями. RSI падает, ADX > 25. Высокая вероятность смены тренда."}

Пример: ETH касается downtrend, объём низкий

LLM: {"score": -3, "explanation": "Пробой на низком объёме (0.6×), линия по 2 точкам. Скорее ложный сигнал — цена вернётся."}

LLM не просто считает — он ДУМАЕТ о контексте: сила линии, объём, рыночная ситуация.
